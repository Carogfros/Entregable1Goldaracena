# SIMULADOR DE REQUERIMIENTO ENERGÉTICO DIARIO 

## Descripción del proyecto
Este simulador permite calcular de forma estimada el requerimiento energético diario
de una persona a partir de datos básicos como edad, peso, altura y nivel de actividad física según el sexo. 


## Funcionamiento
1. El usuario ingresa sus datos personales
2. Se calcula el metabolismo basal utilizando la ecuación de Harris-Benedict, diferenciando entre mujer y hombre. 
3. El metabolismo basal se multiplica por el factor de actividad física.
4. El simulador estima el requerimiento energético diario. 
5. El usuario puede volver a hacer el cálculo o finalizar el simulador.
